Overview

A user-friendly ontology for epigraphic data, extending the CIDOC Conceptual Reference Model (CRM) with human-readable class names aligned to scholarly epigraphic practice.

Select any class or property from the panel to explore its full specification, CIDOC-CRM alignment, EpiDoc mapping, and applicable properties.

Base URI: https://ontology.inscriptiones.org/epont/
Extends: CIDOC-CRM · CRMtex · FRBRoo · SARI Encounter Model
Annotations: SKOS (skos:prefLabel · skos:definition)
